Проект GPT|SOMP3
================

Сайт, вдохновленный Perplexity.ai, с имитацией ИИ-поиска.

Структура:
- index.html - главная страница с поиском
- style.css - все стили сайта
- script.js - логика имитации поиска
- about.html - страница "О проекте"
- topics.html - страница с темами
- assets/ - папка для изображений и иконок

Как запустить:
1. Откройте файл index.html в браузере
2. Введите вопрос в поле поиска
3. Нажмите "Найти ответ"

Функции:
- Имитация ИИ-поиска с ответами
- Указание источников информации
- Адаптивный дизайн для мобильных устройств
- Темная тема с градиентами

Для разработки:
1. Добавьте favicon.ico в папку assets/
2. Замените логотип на свой
3. Добавьте реальный бэкенд для работы с ИИ-моделями
