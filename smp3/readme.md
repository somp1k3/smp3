# SOMP3 Personal Website - PWA версия

Современный персональный сайт-портфолио с поддержкой PWA, модальными окнами и SEO оптимизацией.

## Новые функции

### 1. Модальные окна для "Что я делаю"
- Каждая карточка открывает детальное модальное окно
- Полная информация о каждом направлении деятельности
- Ссылки на соответствующие проекты
- Анимации и плавные переходы

### 2. PWA (Progressive Web App)
- Установка на телефон как веб-приложение
- Работа оффлайн через Service Worker
- Манифест для нативного вида
- Push-уведомления (подготовлено)

### 3. Улучшенный SEO
- Оптимизированные метатеги
- Open Graph разметка для соцсетей
- Twitter Cards поддержка
- robots.txt и sitemap.xml
- Семантическая HTML разметка

### 4. Удален виджет "Мой сайт"
- Упрощенный дизайн
- Более фокус на контенте
- Улучшенная навигация
